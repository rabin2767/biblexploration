# BIBLExploration.com

This section is to load the data to your MySQL DB.

## PREREQUISITE
Install MySQL
Python 3.7 above


### Create Virtual Env
```bash
virtualenv env_<ANY-NAME>
```

### Activate Virtual Env
```bash
.\<PATH To your env_<ANY-NAME>>\Scripts\activate
```

### Deactivate Env
```bash
deactivate
```
